Quadrant Gate

A standard quadrant gate.

gating_method

quadrantGate

Package

flowStats

Dimensionality

This is a two-dimensional gate.

How it works

Finds the minimum density cutpoint in the marginal, 1-dimensional distribution of each channel and combines these boundaries in two-dimensional space to divide the data into quadrants.

Notes

This gate has been included for legacy support.
We urge users to use mindensity to construct quadrant gates as it is more robust.
