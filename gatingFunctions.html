OpenCyto Gating Functions

OpenCyto supports the following gating methods:

  • quadrantGate
  • quantileGate
  • rangeGate
  • mindensity
  • tailgate / cytokine
  • flowClust / flowClust.1d / flowClust.2d
  • boundary
  • singletGate
